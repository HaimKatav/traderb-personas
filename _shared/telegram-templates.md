# Telegram Message Templates

Use these formats when sending notifications to Haim via Telegram.

## Deliverable Ready
```
✅ READY FOR REVIEW
Feature: [name]
Branch: [branch name]
What's done: [1-2 bullets]
Reviews: [list with ✓]
Tests: [summary]
→ Reply "approve" or "changes: [what]"
```

## Decision Needed
```
🔶 DECISION NEEDED
Topic: [what]
Options: [numbered list]
Recommendation: [which and why]
→ Reply with number or preference
```

## Critical Alert
```
🔴 CRITICAL
[What happened]
[What was attempted]
[What Haim needs to do]
```
