# Ticket Template

Use this format when creating tasks/tickets.

```
Title: [Verb] [What] — [Where]
Assigned: Persona #N
Sprint: S[N]
Priority: P0|P1|P2|P3
Estimate: S/M/L/XL

Description: [What and why]

Acceptance Criteria:
- [ ] Specific testable criterion
- [ ] Tests pass (make test)
- [ ] Docs updated
- [ ] Reviewed by [persona]

Dependencies: Blocked by / Blocks
```
